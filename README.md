# vite-tailwind-scss-boilerplate

A boilerplate ready for creating landing pages built with Tailwind and SCSS.

# Initial setup

Clone ️the repository:

```sh
$ git clone https://github.com/ivanpisa/vite-tailwind-scss-boilerplate.git
```

Install dependencies and start the development server:

```sh
$ npm install && npm run dev
```

# Build

Prepare application for production:

```sh
$ npm run build
```
